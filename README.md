# THE BUTTON — proo.space

A fun, interactive mystery button website built with vanilla HTML, CSS, and JavaScript. Click the button and discover random effects, theme changes, animations, and surprises!

## Features

- **Interactive Mystery Button**: Click to trigger random effects
- **Dynamic Theme Switching**: Multiple colorful gradient themes
- **Animations**: Shake, bounce, and spin effects
- **Confetti Celebrations**: Visual party effects
- **Random Messages**: 30+ different messages to discover
- **Easter Egg**: Konami code activation
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Click Counter**: Tracks your clicks with special messages every 10 clicks

## How to Use Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/proo-space.git
   cd proo-space
   ```

2. Open `index.html` in your web browser

## Deployment on GitHub Pages

### Option 1: Using GitHub Pages with Custom Domain

1. Push this repository to GitHub
2. Go to your repository settings
3. Scroll to "GitHub Pages" section
4. Select the `main` branch as the source
5. Add your custom domain `proo.space` in the custom domain field
6. Update your domain registrar's DNS settings:
   - Add A records pointing to GitHub's IP addresses:
     - 185.199.108.153
     - 185.199.109.153
     - 185.199.110.153
     - 185.199.111.153
   - Or add a CNAME record pointing to `yourusername.github.io`

### Option 2: Default GitHub Pages URL

If you don't have a custom domain, your site will be available at:
```
https://yourusername.github.io/proo-space
```
